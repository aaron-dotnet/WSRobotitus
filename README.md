# WSRobotitus

> [!NOTE]
> Early Development

Esto es un web-scraper de la página Robotitus (robotitus.com) a fin de crear una Desktop app portable usando GTK.

## Objetivo

Crear una app portable. He notado que solo es posible leer el contenido de Robotitus mediante la página web y la aplicación para celular, pero no una version Desktop, que es donde paso la mayor parte de mi tiempo.

Es posible crear web-apps en la mayoria de navedadores, pero mi objetivo es mejorar la experiencia de usuario y que sea portable.

## ¿Por qué?

Robotitus es una página de divulgación científica pensada para el público hispanohablante, gracias a su creador `Aldo Bartra` ha permitido que cientos de miles de personas acceder a información de calidad, para mí ha sido de los mejores creadores de contenido y le tengo un fuerte apreció. Me gustaria agradecer con este pequeño granito de arena. Además con esto busco ampliar mi repertorio de proyectos públicos.
